# **App Name**: Subh Janamdin

## Core Features:

- Interactive Greeting Reveal: Reveal personalized birthday messages upon user interaction (e.g., click, hover).
- Dynamic Message Generation: AI generates customized motivational and inspiring birthday messages, and decides when/if to add cultural elements to messages with a reasoning tool.
- High-Quality Animations: Incorporate festive, high-quality animations that trigger upon user interactions or message reveals.
- Emoji Integration: Strategically use prominent and relevant emojis to enhance the festive mood and message impact.
- Cultural Theme: Embed Indian cultural elements and motifs in the visual design.

## Style Guidelines:

- Primary color: Marigold (#FFB703) to evoke warmth and celebration.
- Background color: Light gold (#F2E6D1), offering a bright but soft and desaturated backdrop.
- Accent color: Deep red (#A62303) provides a traditional and striking contrast, signifying good fortune.
- Body and headline font: 'Alegreya', serif, bringing an elegant, intellectual, contemporary feel that suits motivational messages.
- Use culturally relevant icons and emojis (e.g., diyas, sweets, traditional patterns).
- Design a clean, engaging layout that guides the user through the interactive elements and message reveals.
- Implement smooth, celebratory animations (e.g., confetti, sparklers) upon message reveal or user interaction.
